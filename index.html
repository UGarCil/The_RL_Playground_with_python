Live Pyxel

Manual

About

LivePyxel is designed to provide a tool for pixel annotation using the live feed of a webcam to expedite the process of building a dataset for instance segmentation and machine learning

The application was originally intended for the gathering and analysis of date taken from steroscopic and microscopic webcams.

To install LivePyxel, clone the GitHub repository, where you will find additional details on how to create an environment to run the program.
